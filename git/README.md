# Configure User Information

Set up your user information globally.

```bash
git config --global user.name "<Your-Full-Name>"
git config --global user.email "<your-email-address>"
```

# Initialize a Repository

Create a new Git repository or reinitialize an existing one in the current directory:

```bash
git init
```

# Clone a Repository

Clone a repository from a remote URL into a new directory:

```bash
git clone https://github.com/<repo-url>
```

Replace <repo-url> with the URL of the repository you want to clone. This command will copy the entire repository to your local machine.

# Check Status

Check the status of your working tree to see which files have been modified, staged, or untracked:

```bash
git status
```
